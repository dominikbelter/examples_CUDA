# CUDA example code in CMake project

Important: before running this example, check your CUDA driver and NVCC installation. Type in the terminal:
  $ nvidia-smi
  $ lsb_release -a
  $ nvcc --version
        
CUDA examples. Run demo programs:
  - demo_CUDA1 - check GPU info
  - demo_CUDA_add - add some numbers on GPU
  - demo_CUDA_add_kernel - add two vectors on GPU
  - demo_CUDA_initialization - add two vectors on GPU and initialize data on GPU
